## 常用数组操作方法

改变原数组

* **push()**
* **pop()**
* **unshift()**
* **suift()**
* **reverse()** : 颠倒数组中元素的顺序
* **splice()**
* **sort()**

---

不改变原数组

* **join()**
* **slice()**
* **cancat()** : 合并数组，一般使用...代替
* **indexOf()**
* **lastIndexOf()**
* **includes()** : 查找数组是否包含某个元素，返回布尔值
* **flat()** : 数组扁平化处理，传入 number 或 Infinity

---

遍历

* **forEach()**
* **map()**
* **filter()**
* **every()**
* **some()**
* **reduce() reduceRight()** : 为数组提供累加器，合并为一个值
* **find() findIndex()** : 找出第一个符合条件的数组成员，并返回该成员
